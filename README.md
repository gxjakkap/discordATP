# DiscordATP

Simple Discord Anti Phishing message bot.

## Prerequisite

* NodeJs 16


## Setup

1. Install Dependencies

```bash
$ yarn install
```

2. Fill in token in credentials-example.js and rename to credentials.js

```js
const token = ""
//const prefix = ""

module.exports = {
    token,
    //prefix
}

```

3. Run the bot

```bash
$ node .
```

4. Invite the bot to the server
